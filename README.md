# Backstage

## About

Backstage is a web app that makes it easy to organise and run musical jam sessions. It is currently under development.

"Backstage" is a working title that may change in future.

## Key Features

### Jam Sessions

- Admins can create jam sessions for specific dates, which performers can create sets in.
- Jam sessions can be locked to prevent new sets or editing existing sets.
- Jam sessions can be hidden to allow admins to construct a session behind the scenes.

### Sets

- Sets can be organised by anyone.
- Each set has songs with slots for each performer. 
- Performers can sign up to particular slots on a song. Slots are either requested (organiser needs to accept the request), or a set can be free-for-all (slots go to whoever requests them first, no approvals).
- Slots can be requested by multiple people so that if someone drops out, the organiser can see who else was interested in that slot and give it to them.
- Set owners can choose to accept song requests from other users.
- Set owners can define a list of collaborators - other users who are able to manage the set list and slot assignments.
- Sets can be hidden from other users if desired. Collaborators on the set will still see it. Admins see all sets, even hidden ones
- Sets can be locked, preventing further sign ups.


### User Profiles

- Performers can fill out their bio to share anything about themselves, such as favourite artists and any musical projects they are involved in.
- Performers can also select which slot types (instruments) they play.
- Performer profiles are shown on a Who's Who page (which can be opted out of if desired)


### Live Jams

The app doesn't just focus on organising sets in advance, it also features a comprehensive live jam management system:

- Performers can check in upon arrival, this highlights their name on the slots they have signed up for and helps organisers know who has arrived.
- Live Admin interface for jam organisers helps them arrange the running order and take ad-hoc sign ups during the jam, without having to take notes manually.
- Advance sets have a health bar which indicates how many slots are filled. A fuller bar means a healthier set with less effort needed to fill remaining slots.
- Advance slots can be edited directly by the jam manager for any last minute changes or drop outs.
- Live display dashboard shows the sets, slots, and running order for the jam session. This is designed to be shown on a monitor or TV, but any admin can also share the dashboard link via a QR code.
